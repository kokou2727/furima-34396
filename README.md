## アプリの目的
### 誹謗中傷のないSNS
昨今、SNS上では賞賛の言葉だけでなく、批判的な言葉も多く見られます。これらを無くして、かつ、人と人との繋がりをより強固にすることを目的としたものです。

具体的な策としてはSNS上でのコミュニケーションをグループ化し、自分が関わりたいと思う人とだけ関われるようにしました。この目的は批判的な感情を持たせないことではなく、その感情を表に出させないことです。
